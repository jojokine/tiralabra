Ohjelman kääntäminen / ajoympäristö:
------------------------------------
Ohjelma kääntyy java-kääntäjällä, ajoympäristönä toimii kuitenkin hyvin myös toteutusympäristönä käytetty netbeans.

Ohjelman ajaminen:
------------------
Ohjelma voidaan ajaa päätteessä purkamalla Tiralabra.zip -tiedosto, menemällä sen sisältämään kansioon dist ja antamalla käsky java -jar Tiralabra.jar

Ohjelman käynnistyessä ohjelma pyytää verkon solmujen määrän, joka syötetään numeroarvona. Tämän jälkeen ohjelma pyytää syöttämään solmujen välisten kaarien painot, jonka jälkeen se etsii lyhimmän reitin, palauttaa reitin pituuden sekä solmut joiden kautta reitti kulkee.
